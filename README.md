## Objetivo
Corrigir um código com erros e aplicar testes automatizados usando **PHPUnit**
e **GitHub Actions**.

## Passos

1. Clone este repositório.
2. Instale as dependências:
```bash
composer install

3. Rode os testes localmente:
vendor/bin/phpunit
• Você verá que os testes falham inicialmente.
3. Corrija o código em src/Media.php para que os testes passem.
4. Faça commit e push das alterações.
5. Verifique no GitHub se o pipeline (Actions) passou com sucesso.
